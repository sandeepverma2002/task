# Express.js App with EJS

This is a full-stack Node.js app using Express and EJS templates.

## Features

- Server-side rendering with EJS
- PostgreSQL database connection
- Google reCAPTCHA for login
- JWT-based authentication

## Tech Stack

- Node.js
- Express
- EJS
- PostgreSQL
- JWT
- Google reCAPTCHA
